# connect

Connects to Kuzzle using the `host` property provided in the [constructor options](/sdk/js/6/kuzzle/constructor/#arguments-default).
Subsequent call have no effect if the SDK is already connected.

## Arguments

```javascript
connect();
```

## Resolves

Resolves if the connection is successful.
